Isn't it about time updates worked for you?

BitBanshee is a novel framework that uses binary differentiation to shrink update size.
The algorithm behind BitBanshee uses a combination of tried-and-true compression algorithms and system-independent bytewise pattern matching to determine and minimize the number of bits in the binary exectuable that need to be changed in order to transform it from one version to another.
As such, in even the most average of cases, BitBansee can be used to decrease the size of a binary update by as much as a factor of 20.
